# Planet Info Panel Textures

Place your custom info panel images in this directory. These images will be displayed in the sidebar when you follow a planet.

## Required Files

The program will look for these files (PNG or JPG format):

- `sun_info.png` or `sun_info.jpg`
- `mercury_info.png` or `mercury_info.jpg`
- `venus_info.png` or `venus_info.jpg`
- `earth_info.png` or `earth_info.jpg`
- `mars_info.png` or `mars_info.jpg`
- `jupiter_info.png` or `jupiter_info.jpg`
- `saturn_info.png` or `saturn_info.jpg`
- `uranus_info.png` or `uranus_info.jpg`
- `neptune_info.png` or `neptune_info.jpg`
- `moon_info.png` or `moon_info.jpg`

## Notes

- PNG format is checked first, then JPG
- Recommended image size: 512x512 or higher (square images work best)
- If an info texture is not found, the program will fallback to the planet's surface texture
- These textures are separate from the planet surface textures used for 3D rendering
